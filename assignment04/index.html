Whack-A-Pikachu

Description:

Welcome to Whack-A-Mole Pikachu Version! This program is written using object-oriented programming. Each of the 9 moles represent an object with its distinct ‘up’ or ‘down’ state. 1 point is rewarded for every Pikachu whacked and you get 3 points if you hit a Celebi! After 30 seconds, the game ends and we can clearly see that Pikachu got hurt because we kept hitting him with a mallet... Fun game tho :)
